# 🌐 Jaiden Brown – Portfolio Website

Welcome to my personal portfolio website.
I’m Jaiden Brown, a junior at Arizona State University studying Human Systems Engineering with a focus in user experience. This site highlights my projects, skills, and interest in building solutions that combine technology with human-centered design.

# 📌 About This Project

I created this portfolio to showcase my work in UX, web development, and 3D modeling. It represents both my technical skills and my approach to design.

My goal was to build a site that feels clean, interactive, and easy to use across different devices. I focused on keeping the layout simple while still making it engaging and visually appealing.

# 🤖 Building with AI – Reflection

Using AI to help me build this website was a really interesting experience. One thing that stood out to me was how well tools like Copilot could understand what I was asking for. In the past, AI felt less consistent, but this time it was much more reliable.

I learned that AI works best as a support tool rather than a replacement for creativity. It helped me build layouts faster and improve my code, but I still needed to guide the process and make design decisions. Overall, it made development more efficient while still allowing me to stay in control of the final product.


# 📣 Learning Reflection

This project helped me understand how AI and web development can be applied to real-world problems. I also saw how important teamwork is when working on technical projects. Hearing different perspectives helped me think about problems in new ways.

My biggest takeaway is that learning technology is not just about coding. It also involves communication, being flexible, and staying open to feedback. I am looking forward to continuing to grow and improve.
